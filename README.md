# 📊 End-to-End Sales Analysis & Forecasting Dashboard

An interactive Business Intelligence dashboard built using **Power BI, SQL, Excel, Power Query, and DAX** to analyze sales performance, customer behavior, and product trends. The project transforms raw sales data into meaningful insights through dynamic visualizations, KPI tracking, and forecasting.

---

## 🚀 Features

- 📈 Interactive Power BI Dashboard
- 🧹 Data Cleaning & Transformation using Excel and Power Query
- 🗄️ SQL Queries for Business Analysis
- 📊 Dynamic KPIs using DAX
- 🌍 Region-wise Sales Analysis
- 📦 Category & Product Performance Analysis
- 👥 Customer Segmentation
- 📅 Monthly & Yearly Sales Trends
- 🔮 Sales Forecasting
- 🎛️ Interactive Filters and Slicers

---
